# Guardian AI - ML модель безопасности полёта дронов

Система машинного обучения для оценки безопасности полёта дронов на основе метеорологических данных и паспортных характеристик.

## Быстрый старт

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск обучения модели

```bash
python train_safety_model.py
```

## Структура проекта

- `train_safety_model.py` - основной скрипт обучения модели
- `data/drone_specs.csv` - паспортные данные дронов (42 модели)
- `data/data.grib` - погодные данные в формате GRIB
- `data/weather.csv` - конвертированные погодные данные (создаётся автоматически)
- `docs/model_training_documentation.md` - подробная документация

## Что делает скрипт

1. Загружает данные о дронах из CSV
2. Конвертирует GRIB файл в CSV (или генерирует синтетические данные)
3. Создаёт обучающий датасет с метками безопасности
4. Обучает XGBoost модель для предсказания индекса безопасности (0-100)
5. Выводит метрики качества и примеры предсказаний

## Результат

Модель предсказывает:
- **safety_index** (0-100) - индекс безопасности полёта
- **safety_class** (green/yellow/red) - класс безопасности

Подробности в [документации](docs/model_training_documentation.md).
